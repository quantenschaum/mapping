# Nautical Charts

Converting chart data from different sources (S57-ENC, GeoJSON) to

- map tiles
  - web view
  - mbtiles
  - sqlitedb
  - gemf
- OSM
- OsmAnd OBFs

:exclamation: This is still very work in progress!

:point_right: [Documentation](http://waddenzee.duckdns.org/download/)
